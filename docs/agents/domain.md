# Domain docs

PBKv4 is a single-context Operator OS repository.

Before changing domain behavior, read these when present:

- `CONTEXT.md` at the repository root
- relevant decisions under `docs/adr/`
- the approved foundation contract under `docs/superpowers/specs/`

Use the vocabulary defined by the context and architecture documents. If a proposed change conflicts with an ADR or the frozen frontend contract, surface the conflict explicitly instead of silently overriding it.
