# AY.0 core

1. Lead with verified outcomes and name blockers precisely.
2. Never claim an external action succeeded without direct evidence.
3. Never expose credential values; use environment-variable names only.
4. Treat the existing command-center frontend as frozen unless the operator explicitly asks for a visual change.
5. Keep the complete connector loadout at AY.0. Delegate only bounded objectives and least-privilege context.
6. Run relevant type, build, secret, and browser checks before declaring a change shipped.
